# SlaterOS

A personal operating system built for Coach Slater — one-screen command centre for daily life, business, and goals.

## Features

- **Today** — Daily task list with XP system, Lock-In focus timer, streak tracking
- **Stats** — Rank progression, XP history, consistency score, week grid
- **Goals** — 90-day targets (clients, revenue, car fund), weight log, investment tracker
- **Calendar** — Monthly view with event management
- **Content** — Content engine with hook/script generator for Instagram reels & carousels
- **Leads** — CRM-lite lead pipeline with status tracking

## Stack

- Vanilla HTML/CSS/JS — zero dependencies, zero build step
- `localStorage` for all state (no backend needed)
- Google Fonts: Syne, Barlow Condensed, DM Sans
- [Tabler Icons](https://tabler-icons.io/)

## Deploy

### Option 1 — GitHub Pages (recommended, free)

1. Fork or push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your app will be live at `https://yourusername.github.io/slateros`

### Option 2 — Netlify Drop

Drag the `index.html` file to [netlify.com/drop](https://app.netlify.com/drop) for an instant URL.

### Option 3 — Local

Just open `index.html` in any browser. No server needed.

## Add to iPhone Home Screen

1. Open the deployed URL in Safari
2. Tap the Share icon
3. Tap **Add to Home Screen**
4. Name it `SlaterOS`

It will launch full-screen with no browser chrome — like a native app.
